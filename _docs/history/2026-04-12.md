### formulation.md 재작성 — §9.2 Loss 정의

기존 `L_pose(x, y) = Σ ‖p̂ᵢ − pᵢ‖²` 단일 항 정의는 DOPE 실제 loss 와 맞지
않았고, coord loss contribution 이 수식에 드러나지 않아 논문 스토리와
disconnect. 다음과 같이 재작성:

- §9.1 Notation 보강: 6 stage belief / affinity head, `B̂_s / Â_s`, GT `B / A`,
  `softargmax` 기반 `p̂ᵢ`, canonical Y=UP `K_3D_i`
- §9.2.1 **DOPE base loss**: `L_belief = Σ_s ‖B̂_s − B‖² / N_b`,
  `L_affinity = Σ_s ‖Â_s − A‖² / N_a`, `L_base = L_belief + L_affinity`
- §9.2.2 **Coord Huber loss** (contribution):
  `L_coord = (1/8) Σ_i Huber_δ(p̂ᵢ − pᵢ)`, δ = 0.03, Sun et al. 2018 인용
  + "49.5% → 62.2% (v8_A)" 효과 명시
- §9.2.3 Synthetic vs Pseudo-real 분리:
  - `L_syn = L_base + λ_coord · L_coord` (λ = 0.005)
  - `L_real = L_belief only` — affinity / coord GT 없음 (self_train.py
    `generate_belief_maps_from_keypoints` 로 belief 만 재구성)
- §9.2.4 Total loss: `L = L_syn(Dₛ) + α · L_real(D̃ᵣ)`, α default 1.0
