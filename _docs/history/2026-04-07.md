## Canonical Filter v2 필터 개발

### 기존 필터의 문제
- A2/B2/C2가 절대 픽셀 threshold (spread_det > 200px² 등) → 카메라 거리/해상도 바뀌면 안 맞음
- 특정 데이터셋(capture0403noapril)에 과적합된 임계값

### 새 필터 설계 (무차원 비율 기반)
- **Core gate: B ∧ C** (A는 auxiliary score로 강등)
- B: Structural Coverage (hull 면적 비율) + Non-Collinearity (공분산 이방성)
- C: Normalized LOO PnP Stability (LOO reproj / projected diagonal)
- A: Flip consistency score (CSV에 기록만, AND에서 제외)
- Optional priors: depth range, tilt range

### 구현
- `scripts/data_prep/canonical_filters.py` — filter_A, filter_B, filter_C, prior_depth, prior_tilt
- `scripts/data_prep/infer_and_filter_v2.py` — 실행 스크립트 (B∧C core, A auxiliary)

### A를 core에서 뺀 이유
- Flip consistency는 모델의 augmentation equivariance를 간접 측정하는 sanity check
- 어두운 팔레트에서 flip inference가 원본과 다르게 나오는 건 당연 (조명/텍스처 비대칭)
- Hard-but-correct 샘플을 버리는 주 원인이었음
- B와 C는 구조/안정성을 직접 측정하므로 pseudo-label qualification에 더 적합

### Threshold 기준
- tau_cov=0.03 (coverage ratio > 3%)
- tau_ani=0.005 (anisotropy > 0.005)
- tau_C=0.10 (normalized LOO reproj < cuboid 대각선의 10%)
- Synthetic val에서 B=98% 통과, C=53% 통과 → C가 주 필터 역할

## Canonical Filter 확정

### 최종 구조

```
Core gate: B ∧ C (A는 auxiliary score)

B: Visible Structural Support
  B1. Span ratio: 장축 방향 검출 범위 / 재투영 범위 (tau > 0.35)
  B2. Endpoint support: 양쪽 끝 지지 여부, 프레임 밖 면제 (tau < 0.10)
  B3. Non-collinearity: 공분산 이방성 (tau > 0.02)

C: Normalized LOO PnP Stability
  LOO reproj error / projected diagonal (tau < 0.10)

A: Flip consistency (auxiliary score, CSV 기록만)
```

### 설계 결정: "좋아 보이는 reject" 분리

사용자가 시각적으로 "좋다"고 판단한 이미지 8장 중 7장이 B에서 탈락.
분석 결과 전부 one-sided support (span ≈ 0.03~0.06, d_left ≈ 0.95).

```
3가지 "좋다" 기준이 다름:
  시각적으로 괜찮아 보이는 샘플      → overlay가 그럴듯
  runtime demo로 쓸 만한 샘플       → 실시간 표시용으로는 OK
  self-training teacher로 안전한 샘플 → far-side에 증거 없이 hallucination 위험
```

B는 세 번째 기준을 지킴. 완화하면 5번/6번 같은 오검출이 다시 섞임.

### Reject 분류

```
BC hard pass          → self-training 사용
one_sided_plausible   → 보류 (2라운드에서 재평가)
clear_bad             → 완전 제외
```

### 핵심 결론

필터 완화가 아니라 모델 개선이 본질.
양쪽 keypoint를 잡도록 데이터/학습 개선 → BC pass 자연 증가 → self-training 선순환.
