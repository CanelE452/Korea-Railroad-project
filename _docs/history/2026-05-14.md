```
Jetson Orin Nano (Ampere) — Tensor Core 있음 → TF32 기준
  Input 448: 1.5 fps (TRT 4.6)    → frame skip 필요

  Input 224: 4.1 fps (TRT 12)     → 권장 (TRT-FP16 변환 시 풍족)
```
