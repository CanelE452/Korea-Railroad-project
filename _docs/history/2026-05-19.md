### 작업 파이프라인 요약

1. **RealityScan photogrammetry OBJ 두 개 import**
   - `C:\Users\minjae\Desktop\laydown.obj` (눕혀서 촬영)
   - `C:\Users\minjae\Desktop\up.obj` (세워서 촬영)
2. **Decimate + loose parts 정리** (laydown 0.2 → 97K verts, up 0.03 → 252K verts → main만 keep)
3. **OBB covariance 좌표축 정렬** (3d-expert agent 솔루션)
   - vertex covariance matrix 3D eigen decomposition
   - projection extents로 길이순 정렬: 짧은→Z, 중간→X, 긴→Y
   - PCA/face normal 자동 정렬 실패 후 이 방법으로 해결
4. **quarter_right_top 분리** (X>0, Y>0 사분면 추출)
5. **Floor cut** (Z<-0.27, texture color로 floor 식별)
6. **Mirror X+Y** → 전체 팔레트 복원
7. **Fill holes (sides=8,20,50)** + Shade Smooth + Remove doubles
8. **anisotropic scale** 적용 → 1.10×1.30×0.12m 정확히
9. **OBJ export + texture 상대 경로화**
