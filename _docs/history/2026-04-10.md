## Structural Loss Ablation 준비 + VP loss Dead-on-Arrival 발견

### 실험 계획 수정 배경
- 당초 "scratch 65ep vs scratch 65ep" 비교를 계획했으나 v8_A (`weights/v9_ablation_A_coord/`) header 확인 결과 **ft 5ep from mixed_v8/ep60, lr=5e-5, 40분** 구조였음 → 모든 신규 ablation도 동일 세팅으로 공정 비교

### Vanishing Point Loss 구현
- 파일: `Deep_Object_Pose/train/geo_loss.py`
- 함수: `_homogeneous_line`, `_vp_loss_one_group`, `vanishing_point_loss`
- 방식: 4개 평행 edge → 4 homogeneous line → `M = Σ l·lᵀ` 3×3 → `eigvalsh` 최소 고유값 = squared residual; object diagonal로 scale-invariant 정규화
- 무한원점 (평행선)도 finite-VP와 통합 처리
- CLI: `train.py --struct_vp` (기본 0.0)
- `StructuralLoss.forward`에서 `pred_kp` 재사용해서 호출
- Sanity test: perfect cuboid vp≈0, 50px displacement vp=2.4e-3, corner swap vp=1.3e-2, gradient finite ✓

### VP loss 실측 → Dead-on-Arrival 판정
- 스크립트: `scripts/data_prep/measure_vp_scale.py` (8 batch, mixed_v8_train)
- 결과:
  ```
                          coord_raw  edge_raw   vp_raw
  v8_A (pred)             0.117      1.90       2.4e-7
  v8_A (GT)               —          —         -7.3e-6 (≈0)
  mixed_v8 base (pred)    —          —          5.1e-7
  ```
- **결론**: belief map MSE가 이미 GT cuboid 구조를 pixel-wise로 복사시키므로, 학습된 모델의 pred는 이미 VP-consistent. 실제 coord error(0.117)는 cuboid 구조를 유지한 채 위치만 밀리는 형태 → VP loss가 catch 할 게 없음.
- λ 계산: vp를 coord의 10% 수준으로 만들려면 `λ_vp ≈ 93`이 나옴 (비현실적) → VP 실험 완전 포기
- memory 기록: `project_vp_loss_dead_on_arrival.md`

### 결론: Structural Loss 라인 **완전 종료**
- **edge loss (v8_B)**: 두 λ 값 모두 v8_A 이하
  - v8_B2 (λ=0.001): C filter가 9→0으로 완전 붕괴 (LOO PnP 불안정화)
  - v8_B3 (λ=0.0005): PnP rate 반토막, avg_det 감소 (keypoint 감지력 약화)
  - 작은 λ가 더 나쁜 반직관적 결과 — 편차일 수 있지만 둘 다 실패한 건 확실
- **VP loss**: 실측으로 dead-on-arrival 확인 (belief MSE가 암묵적 cuboid prior 학습)
- **flip loss**: 이미 v8_D에서 실패 확정
- **edge loss 실패 해석**: belief MSE + coord가 이미 position을 잡고 있는데 edge log-ratio가 "구조 유지" 방향으로 pull → 오히려 기존 peak가 밀려 LOO PnP 불안정 (C 붕괴) 혹은 keypoint 자체가 약해짐 (avg_det 감소)
- **총 insight**: *"Direct coordinate supervision (coord loss) is sufficient and optimal for fine-tuning converged DOPE models. Indirect structural regularization provides no benefit and typically degrades performance, because the base model already implicitly encodes cuboid structure via belief-map matching."* 이 문장은 논문 ablation의 핵심 메시지로 쓸 수 있음.
