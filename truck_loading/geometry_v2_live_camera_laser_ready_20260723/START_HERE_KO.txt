Geometry V2 카메라 + 레이저 실시간 추론
=========================================

현재 컴퓨터에서는 별도 설치 없이 run_live_windows.cmd 를 더블클릭하면 됩니다.

기본 장치:
  카메라: OpenCV camera 1
  레이저: COM6, 115200 baud, L1 채널

모델 입력:
  camera_height_m = 레이저 L1 거리(m)

유효한 레이저값이 없으면 카메라 화면과 센서 원값만 표시하고 추론은
대기합니다. 임의의 높이값이나 보정값은 사용하지 않습니다.

다른 컴퓨터에서 Python 패키지가 없을 때:
  1. setup_live_windows.cmd 실행
  2. 설치 완료 후 run_live_windows.cmd 실행

키:
  ESC 또는 q : 종료
  Space       : 일시정지/재개
  s           : 화면 저장
  r           : 레이저 중앙값 필터 초기화

다른 카메라를 사용하려면 명령 프롬프트에서:
  run_live_windows.cmd --camera-id 0

프레임 로그를 저장하려면:
  run_live_windows.cmd --log-csv captures\live_log.csv
